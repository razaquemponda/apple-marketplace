# Apple Marketplace Demo

A fully functional demo website for selling refurbished Apple products with order management system.

## Features
- Product catalog with iPhones, MacBooks, and accessories
- Condition labeling (Excellent/Good/Eco-friendly)
- Order form with customer details
- Automatic order notifications
- Mobile responsive design

## Setup Instructions

### 1. Get Formspree Account (Free)
1. Go to [formspree.io](https://formspree.io)
2. Sign up for free account
3. Create a new form
4. Copy your form ID
5. Replace `YOUR_FORM_ID` in `index.html` line 112

### 2. Host on GitHub Pages
1. Create GitHub account
2. Create new repository
3. Upload all files
4. Go to Settings â†’ Pages
5. Select "main" branch as source
6. Your site will be live at `username.github.io/repo-name`

### 3. Customize Products
Edit the `products` array in `script.js` to add your actual products, prices, and descriptions.
